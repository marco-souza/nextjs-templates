# Next.js Templates

Repository to keep track of Next.JS presets, ready for development.

## Examples

- [Simple App](./simple-app) - Setup a small project using `prettier`, `husky`, and `@chakra-ui`
- [Firebase App](./firebase-app) - Setup a Firebase Next App using `prettier`, `husky`, and `@chakra-ui` - requires `.env.local` env file.
